# The Illusion of Fatalism: Distinguishing Causal Determinism from Pre-Destination in Complex Systems (MMG-TR-001)

[![License: CC BY 4.0](https://img.shields.io/badge/License-CC%20BY%204.0-blue.svg)](https://creativecommons.org/licenses/by/4.0/)

This repository hosts the **Technical Reports** of the Meaningfulness Media Group (MMG). This series represents an interdisciplinary effort to move the human agency debate from abstract metaphysics into the domain of **Systems Engineering**.

We treat the human mind as a high-resolution, self-modeling computational system and provide the rigorous frameworks required to defend human dignity in an age of hyper-resolution predictive modeling.

### [MMG-TR-002] Functional Agency in Physical Systems
**Focus:** The Logic Layer (Computer Science & Self-Reference)  
**Status:** Published | [DOI: 10.5281/zenodo.18511450](https://doi.org/10.5281/zenodo.18511450)

Provides a formal computational proof for internal agency. Utilizing the **Halting Problem** and **Computational Irreducibility**, it defines the "Incomputability Firewall" of self-modeling systems.
*   **Key Concepts:** Agency Depth ($D_A$), Effective Agency ($A_e$), Model Fidelity ($R_m$), Process Sovereignty.
*   **Takeaway:** "Free Will" is a technical property of systems where the most efficient way to determine the output is to run the internal deliberation to completion.

### [MMG-TR-003] Cognitive Sovereignty in Algorithmic Societies [Forthcoming]
**Focus:** The Application Layer (Sociology & Policy)  
**Status:** Active Draft / RFC (Request for Comments)

Models the modern "Attention Economy" as an **Adversarial Optimization System**. It analyzes how predictive algorithms bypass System 2 deliberation to render human agents computationally reducible.
*   **Key Concepts:** Ontological Harm, Predictive Resonance ($\rho_p$), The Right to Remain Incomputable, Cognitive Sanctuaries.
*   **Takeaway:** Systematic reduction of human predictability is a structural injury to agency; we must codify "Incomputability" as a foundational digital human right.
